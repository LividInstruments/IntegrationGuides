Overview

WingFX for Guitar Wing turns the Guitar Wing into a wireless digital effects rack. The several effects are designed to pair with the Livid Guitar Wings innovative controls so you can get started playing and getting ideas of how to use your Wing. You can even trigger samples and loops for instant accompaniment out of the box. Launch as a standalone software app, or use as a VST or AU plugin in your favorite DAW.

System Requirements

MACINTOSH

  • OS X 10.8 or greater
  • Intel processor

For plug-ins:

  • VST host compatible with VST 2.4
  • AU host
  • 32 or 64 bit versions available

WINDOWS

  • Windows 7 or greater

For plug-ins:

  • VST host compatible with VST 2.4
  • 32 bit version available

Installation

Macintosh

Download and run the installer to install the WingFX.app standalone version into your Applications folder and the plug-ins into your plug-ins folder. If you want to find the plug-in files, they are installed for all users in

  • VST: <hard drive>:/Library/Audio/Plug-Ins/VST
  • AU: <hard drive>:/Library/Audio/Plug-Ins/Components

Windows

Download and run the installer to install the WingFX.app standalone version into your Applications folder and the plug-ins into your plug-ins folder. The standalone application is installed to

  • C:\Program Files\Livid\WingFX.exe

Plugins are installed, by default, to:

  • C:\Program Files\Common Files\VST\WingFX.dll

If you are running a 64 bit operating system, the 32 bit plugins are installed to:

  • C:\Program Files (x86)\Common Files\VST\WingFX.dll

Getting Started

By design, the WingFX standalone is extremely simple to use.
Launch the application and open the audio settings. To prevent possible noisy feedback from default input/output settings, the first time you launch WingFX, audio processing will be off, and you will be presented with the Audio Settings dialog box to select your input and output and desired buffer size and sample rate. Once set, the settings are saved to your preferences and used the next time you launch WingFX. If the saved audio device is not found the next time, you'll be asked to setup your audio input/output again.

Audio Settings

You must configure your audio interface and MIDI controller settings in the standalone version of WingFX. Click the "options" button in the main interface to open the Audio Settings dialog box.

  • Output - the audio device you want to use for audio output. Channel selection is dependent on the audio interface's setup in the operation system or drivers.
  • Input - the audio device you want to use for audio input. Channel selection is dependent on the audio interface's setup in the operation system or drivers.
  • Sample Rate - Sampling rate for processing incoming audio. The available settings depend on the selected audio device.
  • Audio buffer size - This has a direct effect on latency - the delay between when you play your guitar and when you hear the audio processed. A smaller buffer size will result in lower latency, but higher CPU rate, so selecting this number is a matter of finding the right balance.
  • Active MIDI inputs - Select Guitar Wing Controls as your input.
  • MIDI output - Select Guitar Wing Controls as your output to light the LEDs according to function.

It is possible to use other MIDI controllers to control WingFX, however, you will need to set the controller's output (and input) to match the default settings of the Guitar Wing.

The Effects

WingFX presents a simple processing chain, modeled after a simple set of guitar pedals. There are no editing or adjustments beyond the controls offered by the Guitar Wing controller. WingFX is designed for playability and learning your way around the Guitar Wing - it's not designed as a highly-flexible software production tool. The loop player does not get processed by the effects - they are played directly to the output. Available effects include:

  • Pitch Shift
  • Distortion
  • Low Pass Filter
  • Wah (Band-Pass) Filter
  • Lo-Fi (bit reduction)
  • Chopper
  • Delay
  • Reverb
  • Sample and Loop Player

The Loop and Sample Players

The four bottom buttons on the Guitar Wing are dedicated to 4 different sample files for playback. You can turn looping on or off as desired in the WingFX interface, using your mouse. WingFX is pre-loaded with four basic loops, but loading your own loops and samples is simple. Click on the Browse buttons to load a sound file so it can be triggered with one of the four pads. The program accepts WAV, AIFF, MP3 formats. The plugin version is limited to a 20 seconds length per sample pad.

Plug-In Versions

For Macintosh, there are AU and VST plugins available. For Windows, there is only a VST plug-in format. The plug-ins depend on the host's audio input and output settings, and therefore, there is not any audio setup needed for the WingFX plugins.

Depending on the host and operating system, you may or may not be able to get the MIDI output from the plugin directed to your GuitarWing to light the LEDs. Different hosts have different allowances for MIDI output from plugins.

Additional parameters are available in the host that are not available in the GUI for automation and better control over effect settings.
